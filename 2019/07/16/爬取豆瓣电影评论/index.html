爬取豆瓣电影评论

Mads
2019-07-16
Programming
Python, 爬虫

重拾爬虫，尝试写了一个demo爬取豆瓣”两杆大烟枪“的评论，这也是我最喜欢的一部电影之一，用到的模块包括requests，BeautifulSoup，re。

利用google开发工具分析豆瓣网站，发现豆瓣网站html结构基本相同，每页有20条评论，每页的url区别仅在于start后面的数字，每次加20即更新为下一页的url。
