Python常用转义符

Posted by LST on 2019-07-16

对于python常用转义符的一个总结。

\在行尾时作为续行符

\反斜杠符号

\’单引号

\“双引号

\a响铃

\b退格，光标位置回退一位

\n回车，光标在下一行

\r回车，光标位置回退到本行开头

\f换页，用于打印时

\v纵向制表符

\t横向制表符
